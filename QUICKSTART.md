# 🚀 빠른 시작 가이드 (3분 완성!)

## 1️⃣ 터미널에서 다음 명령어를 순서대로 실행하세요

```bash
# 프로젝트 디렉터리로 이동
cd podowon-church

# 의존성 설치 (약 1분 소요)
npm install

# 데이터베이스 설정 (약 10초)
npx prisma generate
npx prisma migrate dev --name init

# 테스트 데이터 생성 (관리자 계정 포함)
npm run seed

# 개발 서버 시작
npm run dev
```

## 2️⃣ 브라우저에서 확인

- 홈페이지: http://localhost:3000
- 관리자 페이지: http://localhost:3000/admin

## 3️⃣ 로그인하기

### 관리자 계정 (모든 권한)
- 이메일: **admin@podowon.org**
- 비밀번호: **1234**

### 일반 사용자 계정
- 이메일: **user@podowon.org**
- 비밀번호: **1234**

## 4️⃣ 관리자 페이지 사용하기

1. 우측 상단 "로그인" 버튼 클릭
2. 관리자 계정으로 로그인
3. 프로필 드롭다운에서 "관리자 페이지" 선택
4. 6개 탭에서 콘텐츠 관리:
   - 📊 **대시보드**: 통계 확인
   - 🖼️ **배너**: 메인 배너 수정
   - 📋 **메뉴**: 네비게이션 관리
   - 📢 **공지**: 공지사항 추가/삭제
   - 🎤 **설교**: 설교 영상 등록
   - 💬 **게시판**: 게시글 관리

## 5️⃣ 주요 기능 테스트

### 배너 변경해보기
1. 관리자 페이지 → 배너 탭
2. 제목 변경: "2025 부활절 특별예배"
3. "저장" 버튼 클릭
4. 홈페이지로 돌아가서 변경사항 확인 ✅

### 공지사항 작성하기
1. 관리자 페이지 → 공지 탭
2. 새 공지사항 작성
3. "추가" 버튼 클릭
4. 홈페이지에서 즉시 확인 가능 ✅

### 게시판 사용하기
1. 상단 메뉴 → 게시판
2. "기도제목" 탭 선택
3. "글쓰기" 버튼 클릭
4. "비밀글로 작성" 체크
5. 작성 완료
6. 다른 계정으로 로그인하여 "비밀글입니다" 표시 확인 ✅

## ❓ 문제 해결

### "Cannot find module" 오류
```bash
npx prisma generate
```

### "Port already in use" 오류
```bash
PORT=3001 npm run dev
```

### 데이터베이스 초기화
```bash
rm -f prisma/dev.db
npx prisma migrate dev --name init
npm run seed
```
